powpowerman.knownguid module#

既知の電力設定GUIDを提供します。

GCの管理するオブジェクトを減らすため、このファイルでは定数を静的メソッドとして実装します。

class powpowerman.knownguid.PowerKnownSettingGuid#

ベースクラス: object

既知の電力設定のGUID。

static adaptivepowerbehavior_input_controller_state() → Guid#

入力コントローラーのグローバルシステム状態。

static adaptivepowerbehavior_non_adaptive_input_timeout() → Guid#

"ユーザー不明状態の入力タイムアウト秒数。0は無効化を示します。

static disk_adaptive_powerdown() → Guid#

OSがディスクの電源オフに適応型タイマーが使用するか。

static disk_burst_ignore_threshold() → Guid#

ディスク待機時間の計算時に無視する連続アクティブ時間。

static disk_idle_timeout() → Guid#

IO結合の設定を考慮して、最後のディスクアクセスからディスクの電源を切るまでのミリ秒。

static disk_max_power() → Guid#

"最大電力消費レベル。

static disk_nvme_noppme() → Guid#

NVMeのNon-Operational Power State Permissive Modeが有効か。

static disk_powerdown_timeout() → Guid#

最後のディスクアクセスからディスクの電源を切るまでの秒数。

static idleresidency_deep_sleep_enabled() → Guid#

ディープスリープポリシー設定。

static idleresidency_disk_coalescing_powerdown_timeout() → Guid#

IO結合がアクティブな場合に最後のディスクアクセスから電源を切るまでのミリ秒数。

static idleresidency_execution_required_request_timeout() → Guid#

CS Enterから実行要求リクエストを非活性化するまでの待機秒数。

0：実行電力要求は無効化されています。

-1：実行電力要求は常に活性化されています。

static sleep_allow_awaymode() → Guid#

退席モードが有効か。

static sleep_critical_power_transition() → Guid#

スリープ状態移行の切迫性制御。切迫スリープ移行はアプリケーション、システム、ドライバーに通知されません。

static sleep_hibernate_timeout() → Guid#

ハイバネーション(S4)までの待機秒数。

static sleep_idle_threshold() → Guid#

待機閾値%。Windows 7 (Windows 6.1)で廃止されました。

static sleep_standby_budget_grace_period() → Guid#

Standby Budget Grace Period。

static sleep_standby_budget_percent() → Guid#

Standby Budget Percent。

static sleep_standby_timeout() → Guid#

スタンバイモード（S1、S2、S3）へ移行する前の「待機状態」と判定する秒数。

static sleep_system_awaymode() → Guid#

退席モード中か。

static sleep_unattend_sleep_timeout() → Guid#

システム無人スリープタイムアウト後に再スリープするまでの秒数。 0は標準standby/hibernate待機ポリシーの使用を示します。

static sleep_user_presence_prediction() → Guid#

User Presence Predictionモードが有効か。

static video_adaptive_display_brightness() → Guid#

OSが画面の適応型調光に環境光センサーを使うか。

static video_adaptive_percent_increase() → Guid#

適応型調光タイムアウトの増加量。Windows 8.1で廃止されました。

static video_adaptive_powerdown() → Guid#

OSが画面の電源オフに（以前の動作に基づく）適応型タイマーを使うか。

static video_advanced_color_quality_bits() → Guid#

Advanced Color対応ディスプレイで視覚効果とバッテリー寿命のどちらを優先するか。

static video_allow_display_required() → Guid#

電力要求に必要なディスプレイを作成可能か。

static video_annoyance_timeout() → Guid#

適応型調光のオン・オフ。Windows 8.1で廃止されました。

static video_console_display_state() → Guid#

現在のモニター状態。

static video_console_lock_timeout() → Guid#

インタラクティブコンソールのロックから（センサーがユーザー不在を示してから）画面の電源を切るまでの秒数。0は無効化を示します。

static video_current_monitor_beightness() → Guid#

現在のモニターの明るさ。

static video_devicepowerpolicy_brightness() → Guid#

通常状態の画面の明るさポリシー。

static video_devicepowerpolicy_dim_brightness() → Guid#

調光状態の画面の明るさポリシー。

static video_dim_timeout() → Guid#

ユーザーの最終入力から画面を暗くするまでの秒数。

static video_hibernate_fasts4_policy() → Guid#

高速ハイバネーション(S4)対応時に有効か。

static video_monitor_power_on() → Guid#

モニターの電力がオンか。

static video_powerdown_timeout() → Guid#

ユーザーの最終入力から画面の電源を切るまでの秒数。

class powpowerman.knownguid.PowerKnownSubgroupGuid#

ベースクラス: object

電力サブグループの名前は電力サブグループのプレフィックスで使用するため、 名前には長くなりすぎないようにハイフンは用いません。
