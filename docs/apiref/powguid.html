class powguid.Guid#

ベースクラス: Structure

GUID型。

サンプル

>>> iid_iunk1 = Guid.from_define(0x00000000, 0, 0, 0xC0, 0, 0, 0, 0, 0, 0, 0x46)
>>> iid_iunk2 = Guid.from_str_d("00000000-0000-0000-C000-000000000046")
>>> iid_iunk3 = Guid.from_str_b("{00000000-0000-0000-C000-000000000046}")
